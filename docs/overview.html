GOG Galaxy - Community Integration - Python API¶

The API allows to integrate community plugins for external gaming platforms with GOG Galaxy 2.0. The provided features are:

  • multistep authorisation using a browser built into GOG Galaxy 2.0

  • cache storage

  • easy deployment with the a Python interpreter built into GOG Galaxy 2.0

  • GitHub-integrated autoupdate system

  • ability to implement methods for:

    • importing owned and installed games

    • installing and launching games

    • importing achievements and playtime statistics

    • importing friends

    • receiving and sending chat messages

    • …and more!

Basic usage¶

A plugin should inherit from the Plugin class. Supported methods like get_owned_games() should be overwritten - they are called from the GOG Galaxy client in the appropriate times. Each of those method can raise exceptions inherited from the ApplicationError. Communication between the plugin and the client is also possible with the use of notifications, for example: update_local_game_status().

import sys
from galaxy.api.plugin import Plugin, create_and_run_plugin
from galaxy.api.consts import Platform

class PluginExample(Plugin):
    def __init__(self, reader, writer, token):
        super().__init__(
            Platform.Generic, # Choose platform from available list
            "0.1", # Version
            reader,
            writer,
            token
        )

    # implement methods
    async def authenticate(self, stored_credentials=None):
        pass

def main():
    create_and_run_plugin(PluginExample, sys.argv)

# run plugin event loop
if __name__ == "__main__":
    main()

Deployment¶

The client has a built-in Python 3.7 interpreter, so the plugins are delivered as .py files. The additional manifest.json file is required:

{
    "name": "Example plugin",
    "platform": "generic",
    "guid": "UNIQUE-GUID",
    "version": "0.1",
    "description": "Example plugin",
    "author": "Name",
    "email": "author@email.com",
    "url": "https://github.com/user/galaxy-plugin-example",
    "script": "plugin.py"
}
